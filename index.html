Panduan Koneksi Manual (tanpa login)

Kalau lebih suka sambungkan Power BI langsung ke database, gunakan cara ini.

1
Buka Power BI Desktop
Get Data → PostgreSQL database (untuk Supabase) atau langsung pakai Web connector ke URL API ini.
2
Gunakan URL API
Masukkan URL berikut sebagai Web data source di Power BI:
https://your-app.railway.app/api/studi-kasus
3
Transform & Load
Power BI akan parse JSON otomatis. Expand kolom "hasil" untuk mendapatkan ranking dan skor per alternatif.
